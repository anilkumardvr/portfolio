# Anil patch for Sumanth Netflix portfolio repo

Copy these files into your cloned `anil-portfolio-clone` folder, replacing existing files.

Then run:

```bash
cd ~/Desktop/anil-portfolio-clone
npm install
npm start
```

This patch:
- Keeps Sumanth repo animations/GIF-style layout
- Replaces DatoCMS calls with local Anil data
- Replaces name, profile banner, contact info, skills, timeline, projects
- Replaces avatars and intro sound
